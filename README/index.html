About

This project was created by Ann Farnsworth-Alvear and staff in the Research Data and Digital Scholarship department at Penn Libraries. The site's code is licensed under CC-BY-NC-4.0.

Under the hood

This project was created using Eleventy. It utilizes Decap CMS for easy content addition. The design is based on the template AgriCulture with modifications. It may have additional software dependencies.
